## Project Description

This is my first web project. It allows users to search for cars, view detailed information about them, and find cars that match their preferences through a tailored questionnaire. Whether the user knows exactly what car they're looking for or needs guidance, the platform helps them discover the perfect vehicle.


## Features

- **Car Search**: Search for any car model and view detailed specifications, features, and images.
- **Questionnaire**: A 10-question survey matches users with suitable car models based on their preferences.
- **Car Visualization**: Visual representations and images of cars give users a better idea of the vehicle's appearance.

## Technologies Used

### Frontend:
- React
- HTML
- CSS
- Tailwind CSS

### Backend:
- **APIs**: The application uses external APIs to fetch car data and images.
  - **RapidAPI**: For car specifications and features.
  - **CarImage API**: For retrieving car images based on the selected manufacturer, model, and year.
- **Environment Variables**: API keys and URLs are stored securely in a `.env` file.
